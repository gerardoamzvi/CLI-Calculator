#  CLI Calculator in C | calc

A simple command-line calculator built in C, as a learning project to discover C fundamentals (pointers, strings, CLI arguments, REPL).

## Usage

```bash
# Single expression
./calc 3 + 5

```

## Build

Build the calc.exe file
```bash
gcc main.c -o calc 
```

## Launch
To run the executable on windows 

```bash
# Single expression
./calc 3 + 5

```
## Features

- Basic operations: `+` `-` `*` `/`
- Decimal numbers
